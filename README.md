# 📊 1. State of Data Brasil 2023 — Análise do Mercado de Dados

## 🎯 Objetivo

Análise exploratória do dataset **State of Data Brasil 2023**, com foco no perfil de profissionais da área de dados no Brasil.

O projeto investiga padrões relacionados a:

- Gênero
- Idade
- Escolaridade
- Região
- Áreas de atuação (Data Science, Data Engineering, Data Analytics)

---

## ⚙️ Pipeline de Análise

- Importação e leitura do dataset (CSV)
- Tratamento de valores ausentes e inconsistentes
- Renomeação e padronização de colunas
- Filtragem e criação de subconjuntos analíticos
- Visualização de distribuições e comparações
- Extração de insights

---

## 🧰 Tecnologias

- Python 
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook

# 📍 2. Análise de Dados — Brazil Cities Dataset

## 🎯 Objetivo

Exploração e análise de dados de cidades brasileiras com foco em:

- Estruturação e organização dos dados
- Limpeza e tratamento
- Análise exploratória (EDA)
- Visualização de padrões

---

## ⚙️ Etapas do Projeto

- Importação do dataset
- Limpeza e tratamento de dados
- Seleção e transformação de variáveis
- Análise exploratória
- Visualização de dados
- Interpretação de resultados

---

## 🧰 Tecnologias

- Python 
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 📊 Resultados

- Identificação de padrões relevantes nos dados  
- Visualizações claras e interpretáveis  
- Insights estatísticos consistentes  
- Preparação de dados para análises futuras
